01 有个莲花池里起初有一只莲花，每过一天莲花的数量就会翻一倍。假设莲花永远不凋谢，30天的时候莲花池全部长满了莲花,
   请问第23天的莲花占莲花池的几分之几？
   首先定义最终终止条件f(1)=1; 
   然后定义递归公式中f(n)=f(n-1)*2。
   
   int fn(int day){
		if (1 == day){
			return 1;
		}

		day = fn(day - 1) * 2;

		return day;
	}

	void test(){
		int a = fn(23);
		int b = fn(30);
		double c = (double)a / b;
		printf("%f\n", c);
	}
